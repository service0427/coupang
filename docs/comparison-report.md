### 예상과 다른 결과

2. **추적 픽셀**: Mercury 추적 픽셀(200개 이상)은 항상 새로 로드됨
3. **캐시 정책**: 많은 리소스가 no-cache 또는 짧은 만료 시간 설정

1. **리소스 차단이 더 효과적**
   - 캐시보다는 불필요한 리소스 차단이 더 큰 절감 효과
   - 특히 추적 픽셀과 광고 차단 필요

3. **캐시 활용 한계**
   - 동적 콘텐츠가 많은 이커머스 사이트 특성상 캐시 효과 제한적
